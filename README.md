## About the Project 📂
This application contains details of some books in it. We can add any new book, modify any existing book or can delete any book also. Mysql is used for the database and cloudinary is used to store cover images of the books. 
The backend part of this application is present in different repository. You can access it from here - 


👉 <a href='https://github.com/alok-96/Book-Store-Backend'>Book-Store-Backend</a>

### Built With:

» React.js <br />
» CSS <br />
» Node.js <br />
» Express.js <br />
» MySql <br />

## Installation and Setup Instructions 🛠️

```
1. Clone this frontend repository
   git clone https://github.com/alok-96/Book-Store-Frontend.git
    
2. Go into the repository
   cd Book-Store-Frontend

3. Install dependencies
   npm install

4. Start a local dev server
   npm start

5. Clone the backend repository
   git clone https://github.com/alok-96/Book-Store-Backend.git
    
6. Go into the repository
   cd Book-Store-Backend

7. Install dependencies
   npm install

8. Start a local dev server
   nodemon
```
